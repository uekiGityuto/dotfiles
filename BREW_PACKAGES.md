# Homebrew パッケージ一覧

## バージョン管理

| パッケージ | 用途                                                                                           |
| ---------- | ---------------------------------------------------------------------------------------------- |
| mise       | 各種言語/ツールのバージョン管理（Node, Ruby, Flutter, Java, Python, Go, Terraform, pnpm など） |

## CLI ツール

| パッケージ | 用途                                         |
| ---------- | -------------------------------------------- |
| coreutils  | GNU 版コマンド（Linux 互換の ls, cp, mv 等） |
| fzf        | ファジーファインダー（インタラクティブ検索） |
| gh         | GitHub CLI                                   |
| git-lfs    | Git Large File Storage（大容量ファイル管理） |
| ripgrep    | 高速 grep（コード検索）                      |
| jq         | JSON パーサー                                |
| yq         | YAML パーサー                                |
| tree       | ディレクトリ構造表示                         |
| wget       | ファイルダウンロード                         |
| zoxide     | スマートな cd（ディレクトリ移動の高速化）    |

## ターミナル環境

| パッケージ | 用途                                     |
| ---------- | ---------------------------------------- |
| zellij     | ターミナルマルチプレクサ（画面分割）     |
| yazi       | ターミナルファイラー（プレビュー付き）   |
| lazygit    | Git TUI クライアント（GUI 並みの操作性） |

## AI CLI

| パッケージ | 用途              |
| ---------- | ----------------- |
| gemini-cli | Google Gemini CLI |

## クラウド/インフラ

| パッケージ | 用途             |
| ---------- | ---------------- |
| awscli     | AWS CLI          |
| tflint     | Terraform Linter |

## Linter/品質ツール

| パッケージ | 用途                    |
| ---------- | ----------------------- |
| shellcheck | シェルスクリプト Linter |
| hadolint   | Dockerfile Linter       |
| yamllint   | YAML Linter             |

## データベース

| パッケージ   | 用途                        |
| ------------ | --------------------------- |
| mysql-client | MySQL クライアント          |
