# Homebrew パッケージ一覧

## バージョン管理

| パッケージ | 用途 |
|-----------|------|
| mise | 各種言語/ツールのバージョン管理（Node, Ruby, Flutter, Java, Python, Go, pnpm） |

| パッケージ | 用途 |
|-----------|------|
| coreutils | GNU版コマンド（Linux互換のls, cp, mv等） |

| git-lfs | Git Large File Storage（大容量ファイル管理） |
| ripgrep | 高速grep（コード検索） |
| jq | JSONパーサー |
| yq | YAMLパーサー |
| tree | ディレクトリ構造表示 |
| wget | ファイルダウンロード |
